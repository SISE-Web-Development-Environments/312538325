About Me


My name is Sara Shamai, but everyone call me Saly.

I was born in 1953 in Rishon Lezion.

I attended the Kannot boarding school and from there I joined the army to serve in the "Nahal" brigade.

We were settling and setting up new kibbutzim, and so me and other members of the "Garin Nahal" came to live at Kibbutz Givat Haim.

I like to paint and I go to a drawing class once a week. I also like to do walks with friends around the kibbutz.

In my spare time, I love working in the garden and planting flowers.

Three main differences between web 1.0, web 2.0 and web 3.0

web 1.0 web 2.0 web 3.0

General Explanation

Web 1.0 refers to the first stage of the World Wide Web evolution. Earlier, there were only few content creators in Web 1.0 with the huge majority of users who are consumers of content. Personal web pages were common, consisting mainly of static pages hosted on ISP-run web servers, or on free web hosting services. Web 2.0 refers to world wide website which highlight user-generated content, usability and interoperability for end users. Web 2.0 is also called participative social web. It does not refer to a modification to any technical specification, but to modify in the way Web pages are designed and used. web 3.0 refers the evolution of web utilization and interaction which includes altering the Web into a database. In enables the upgradation of back-end of the web. Web 3.0 is a term which is used to describe many evolutions of web usage and interaction among several paths. In this, data isn’t owned but instead shared, where services show different views for the same web / the same data.
